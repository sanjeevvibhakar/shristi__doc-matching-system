HOW TO RUN LOCALLY USING XAMPP:

1. Copy the entire 'organized_project' folder to your XAMPP's htdocs directory.

2. Start Apache and MySQL using XAMPP Control Panel.

3. Make sure you create a database and required tables in phpMyAdmin matching your code (you might need to adjust 'uconfig.php').

4. Open browser and navigate to:
   http://localhost/organized_project/login.php

5. From there you can register a new user, login, upload documents, etc.
